Metamodel files
---------------
 * MSOfficeExcel_SpreadsheetMLSimplified.ecore
 * MSOfficeExcel_SpreadsheetMLSimplified.km3
 * SoftwareQualityControl.ecore
 * SoftwareQualityControl.km3

Models' examples files
-----------------------
 * BugTrackingXMLExample.xml: Excel XML file example (directly openable with Excel 2003)
 * BugTrackingSpreadsheetMLExample.ecore: the result of the injection of this XML file into a a SpreadsheetMLSimplified model.
					(see the "Microsoft Office Excel Injector" transformation...)
 * SoftwareQualityControlExample.ecore: the generated SoftwareQualityControl model 

Transformation files
--------------------
 * SpreadsheetMLSimplified2SoftwareQualityControl.atl


"MSOfficeExcel_SpreadsheetMLSimplified.ecore" and "SoftwareQualityControl.ecore" are the metamodels for SpreadsheetMLSimplified and SoftwareQualityControl models.
Km3 files provide readable versions of these metamodels in the km3 format.
