Metamodel files
---------------
 * MSOfficeExcel_SpreadsheetMLSimplified.ecore
 * MSOfficeExcel_SpreadsheetMLSimplified.km3
 * Table.ecore
 * Table.km3
 * XML.ecore
 * XML.km3

Models' examples files
-----------------------
 * TableModelExample.ecore: Table model
 * exampleTable.html: corresponding table in HTML format
 * ExcelModelExample.ecore: Excel SpreadsheetMLSimplified model
 * XMLModelExample.ecore: XML model of the Excel workbook
 * exampleExcelJavaSource.xml: corresponding XML text 

Transformation files
--------------------
 * Table2SpreadsheetMLSimplified.atl
 * SpreadsheetMLSimplified2XML.atl
 * XML2ExcelText.atl


"MSOfficeExcel_SpreadsheetMLSimplified.ecore", "Table.ecore" and "XML.ecore" are the metamodels for SpreadsheetMLSimplified, Table and XML models.
Km3 files provide readable versions of these metamodels in the km3 format.
