Models' examples files
-----------------------
 * SoftwareQualityControlExample.ecore: the input SoftwareQualityControl model
 * BugzillaExample.ecore: the corresponding Bugzilla model  
 * BugzillaXMLExample.ecore: the corresponsding XML model
 * BugzillaXMLfileExample.xml: the generated output Bugzilla XML file  

Transformation files
--------------------
 * SoftwareQualityControl2Bugzilla.atl
 * Bugzilla2XML.atl
 * XML2BugzillaText.atl


"SoftwareQualityControl.ecore", "Bugzilla.ecore" and "XML.ecore" are the metamodels for SoftwareQualityControl, Bugzilla and XML models.
Km3 files provide readable versions of these metamodels in the km3 format.
