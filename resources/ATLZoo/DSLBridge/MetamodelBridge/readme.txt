Example files
-------------
 * SimpleExample/DSL2EMF/SimpleExampleMM.xml: the DSL metamodel xml file exported (as exported by DSLTools)
 * SimpleExample/DSL2EMF/SimpleExampleMM-XML.ecore: the XML model corresponding to this XML file
 * SimpleExample/DSL2EMF/SimpleExampleMM-DSL.ecore: the DSL model generated from the previous XML model
 * SimpleExample/DSL2EMF/SimpleExampleMM-KM3.ecore: the KM3 model generated from the previous DSL model
 * SimpleExample/DSL2EMF/SimpleExampleMM-KM3.km3: the KM3 file corresponding to the previous Km3 model

 * SimpleExample/EMF2DSL/SimpleExampleMM-KM3-2.km3: new version of the KM3 file obtained with the DSL2EMF
 * SimpleExample/EMF2DSL/SimpleExampleMM-EMF-2.ecore: the EMF model corresponding to this KM3 file
 * SimpleExample/EMF2DSL/SimpleExampleMM-KM3-2.ecore: the KM3 model corresponding to this KM3 file
 * SimpleExample/EMF2DSL/SimpleExampleMM-DSL-2.ecore: the DSL model generated from the previous KM3 model
 * SimpleExample/EMF2DSL/SimpleExampleMM-XML-2.ecore: the XML model generated rom the previous DSL model


Transformation files
--------------------
 * XML2DSL.atl
 * DSL2KM3.atl

 * KM32DSL.atl
 * DSL2XML.atl
 * XML2Text.atl

The XML2DSL transformation ("XML2DSL.atl") can be tested with the XML model ("SimpleExample/DSL2EMF/SimpleExampleMM-XML.ecore") as input.
The source XML model is obtained by injecting the initial xml file ("SimpleExample/DSL2EMF/SimpleExampleMM.xml") into an XML model.
This transformation returns a DSL model ("SimpleExample/DSL2EMF/SimpleExampleMM-DSL.ecore").

The DSL2KM3 transformation ("DSL2KM3.atl") can be tested with the DSL model ("SimpleExample/DSL2EMF/SimpleExampleMM-DSL.ecore") as input.
This transformation returns a KM3 model ("SimpleExample/DSL2EMF/SimpleExampleMM-KM3.ecore").
This KM3 model can be injected into a KM3 file ("SimpleExample/DSL2EMF/SimpleExampleMM-KM3.km3").


The KM32DSL transformation ("KM32DSL.atl") can be tested with the KM3 model ("SimpleExample/EMF2DSL/SimpleExampleMM-KM3-2.ecore") as input.
The KM3 source model is obtained by injecting the new KM3 file ("SimpleExample/EMF2DSL/SimpleExampleMM-KM3-2.km3") into a KM3 model.
This KM3 file is a different version of the file generated by the DSL2EMF bridge, in which some features that may raise errors with some version of ATL tools with have been corrected.
The EMF model (SimpleExample/EMF2DSL/SimpleExampleMM-EMF-2.ecore) is obtained by injecting this KM3 file into an EMF model.
This transformation returns a DSL model ("SimpleExample/EMF2DSL/SimpleExampleMM-DSL-2.ecore").

The DSL2XML transformation ("DSL2XML.atl") can be tested with the DSL model ("SimpleExample/EMF2DSL/SimpleExampleMM-DSL-2.ecore") as input.
This transformation returns a XML model ("SimpleExample/EMF2DSL/SimpleExampleMM-XML-2.ecore").
This XML model can be extracted into a xml file using the query XML2Text.atl.
