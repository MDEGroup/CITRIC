Models' examples files
-----------------------
 * SoftwareQualityControlExample.ecore: the input SoftwareQualityControl model
 * MantisExample.ecore: the corresponding Mantis model  
 * MantisXMLExample.ecore: the corresponsding XML model
 * MantisXMLfileExample.xml: the generated output Mantis XML file  

Transformation files
--------------------
 * SoftwareQualityControl2Mantis.atl
 * Mantis2XML.atl
 * XML2MantisText.atl


"SoftwareQualityControl.ecore", "Mantis.ecore" and "XML.ecore" are the metamodels for SoftwareQualityControl, Mantis and XML models.
Km3 files provide readable versions of these metamodels in the km3 format.
